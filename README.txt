Demo code for 1.3inch LCD display HAT for Raspberry Pi.
The display is based on ST7789 controller ([Datasheet](https://www.waveshare.com/w/upload/a/ae/ST7789_Datasheet.pdf)).

Sources in raspberry-pi/ is a slightly enhanced version of official demos
downloaded from: https://www.waveshare.com/wiki/1.3inch_LCD_HAT

Directory pic32/ contains a version for Microchip PIC32MZ DA Starter Kit
running Linux.
